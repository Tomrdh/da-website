What is Discord Addons?

Discord Addons (by TomRdh) is a project containing themes (and maybe plugins later) for Discord. It can be used directly on Discord by using a client mod, or on browser by installing an extension.

The files in this project are hosted on GitHub. View source.

How is it organisated?

If you don't know how repositories works on GitHub, it's just literally files and folders, just like your file explorer.

Some addons are called "Themes", it's themes for Discord, and some others are called "Styles", it's styles on browser.


The themes can be installed in the download-themes-here folder in GitHub or in the Themes page in this website.

For styles, it's in the other folder in GitHub and in the Other page here.

But I recommand you to use the navigation buttons and download themes and styles directly in this website, it's easier. If you need more help, you can read this wiki.

Note

Tomrdh/discord-addons is generated from YTGamer/bdthemes to continue updating these themes. YTGamer/bdthemes is now discontinued.
