1. Download a theme

Use ONLY ONE source for downloading. You DON'T need to use everything, only for one theme file.

1.1. Download a theme using GitHub

First, you must know that browsing files on GitHub is just like browsing files on explorer, but for hosted files. Then you can follow these steps:

  1. First, make sure you're on the Tomrdh/discord-addons repository, and not another one (like a fork).
  2. Go to the download-themes-here folder.
    • If your window/screen width is small, you'll have to click on "View code" before.
  3. Each folder has the name of the theme itself. Click on the one you want.
  4. You can now follow the "Apply a theme on [...]" tutorial.

1.2. Download a theme using this website

  1. Go to the themes page. You can also find it from the main page, it's the "View themes" button.
  2. Choose the theme you want. You can hold on an image to resize it.
  3. Click on the download button below.
    • For GitHub download buttons, it redirects you to the right theme folder.
    • For BetterDiscord's website links, it redirects you to the right theme page.
    • You can look at yourself if there are other buttons available, but it's not required.
  4. You can now follow the "Apply a theme on [...]" tutorial.

1.3. Download a theme using the BetterDiscord's official website

Downloading a theme from that website is ridiculously easy. Just go to the theme's page, then you can follow the "Apply a theme on [...]" tutorial.

Only official themes are listed on that website, so you won't find every Discord Addons themes, but you can find all other official ones, which are also good.

2. Apply a theme

The tutorial for applying a theme is different for each client modification. We won't explain you how to install a client mod, please use official websites for that.

All tutorials about applying a theme consider that you're already in the right GitHub folder to get the theme you want.

2.1. Apply a theme on BetterDiscord

You probably know the most known one, so this won't be long, and it's very simple, because it's just moving a file in a folder.

  1. Once you found the theme you want (meaning that you're in the right folder on GitHub or the right page on BD), download only the file ending with .theme.css by clicking on it, next "Raw", and then Ctrl + S.
  2. On BetterDiscord's themes settings, click on "Open Themes folder". If it doesn't open, try opening %appdata%\betterdiscord\themes (\plugins for plugins).
  3. Move the file that you downloaded there. And obviously, enable it!

2.2. Apply a theme on Stylus

Note that Stylus is an extension for browser! Not a real client modification! May not work with Discord on Firefox.

  1. Once you're in the right folder in GitHub, click on the file ending with .theme.css, then "Raw".
  2. Select everything and copy. You don't need to save the file, like with BetterDiscord.
  3. Click on the Stylus extension icon, then "Manage".
  4. On the sidebar on the left, click on "Write new style". Don't check "As Usercss"!

  5. Give a name for the style. In the code part, paste everything.
  6. At the bottom, don't forget to apply only URLs on the domain: discord.com.

  7. Save, and it should be applied!

2.3. Apply a theme on GooseMod

Oh Jesus, on GooseMod it's really easy, but too easy to explain... Just grab the theme from the modules store!

3. Change the background, colors and other things with Opera GX Gaming Theme

It's very easy! That's just changing some variables. If you're on BetterDiscord or Powercord, you'll have to open the theme file where you saved it. On Stylus, just click on the edit button.

Once you opened it, you'll see a lot of variables and values. You don't need to change everything to change the whole theme.

3.1. Change the background

There's a --background variable, that should be the first one. By default, its value is var(--opera-gx-bg-1). You can change this value to:

  • Another preset background. The default one is var(--opera-gx-bg-1), so you'll just have to replace the 1 with another number to see the other preset backgrounds. (eg: var(--opera-gx-bg-15))
  • A custom background. This time, delete the whole value, and replace it with url(). In these parentheses, you'll have to put a direct image link, so you'll have to upload it where you want. Once you did it, right click the image and copy image address. And finally, paste this address in these parentheses, then save. (eg for imgur: url(https://i.imgur.com/ABcd012.png))
  • Just a color. Why not? You can use any format you want, like using another variable for this one, a gradient, or even transparent if you want your BetterDiscord transparent (read more about this...). (eg for colors: rgb(255, 0, 128)) If you're not familiar with using numbers to set a color, you can try this color picker.

3.2. Change the colors and transparency

Color variables are RGB only (Red-Green-Blue - three numbers between 0 and 255), and transparency ones are only between 0 and 1.

You can use other variables to set multiples ones, to avoid some kind of copy-pasting. The format is var(--varname), and replace varname with the name of the variable you want.


Between the background ones, you can see a lot of variables, you can change them like you want, following CSS "rules". You don't need to change everything, unless you know what you're doing. Changing only Color-Settings should be enough. If you need help about each variable, you can read the comment next to the set value.

3.3. Use an available preset (NEW)

You have to go in the presets subfolder in GitHub at the top of the files (or by following this path: download-themes-here --> OperaGxGamingTheme --> presets), then download the ones you want.

3.4. Submit your own preset (NEW)

Not here yet, but will be available soon. You can start creating your presets from now on, then wait when submitting will be available and announced. Custom presets will be credited by yourself and the theme's author, and will be manually verified. Small changes could be applied without changing the appearance.

4. Remove lines between messages with Opera GX Gaming Theme

There are 2 ways to remove these lines between messages. The first one is setting the message spacing to 0 or 4px from Discord's Appearance settings.


The second way, open the theme file, and paste this code at the very bottom:

.groupStart-23k01U {
    border-top: none !important;
}

5. Make BetterDiscord transparent with Opera GX Gaming Theme

This works only with BetterDiscord, and any other client modification that supports a transarency option, which must be enabled. Also, Discord must be fully relaunched if you just enabled that right now (BetterDiscord should prompt you a confirmation).


If you're ready to make this theme transparent, then first open the theme file. At the very top of the variables, you can see the Background-Settings. Just set both variables to transparent, and save. If it's required, reload Discord, or the theme itself.
