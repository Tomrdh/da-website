Other - Discord Addons

Styles to use on Stylus on some other websites. To download and install a theme on Stylus, click on "Download" links, click "Raw", select all (Ctrl + A) copy (Ctrl + C) and paste (Ctrl + V) in a new style. Don't forget to add the indicated domain or you'll have unwanted results in other websites!

Diep.io Dark Neon Theme

A dark theme for diep.io. To use this script, use the Home key on your keyboard (may be a top-left arrow) to open the in-game console, and paste the script. Hit Enter and press Home again to close the console.

  • Note 1: If you leave/reload the game, you must paste the code again.
  • Note 2: This theme also allows you to view the FPS counter and the exact value of the health of an element. (but it's small…) (very small…) (sorry, not my fault)
